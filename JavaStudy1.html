2 Java的语法
2.1 1.Java的变量

类型 名称 = 变量值;

(1)变量的类型

基本数据类型：4类8种

①整型：byte(1),short(2),int(4)默认类型,long(8)

   定义long：long num = 12L;

②浮点类型：float(4),double(8)默认类型

   定义float：float f = 12.0f;

③布尔类型：boolean true,flase

④字符类型：char

   字符类型不是字符串，用单引号定义，而且只能有一个字符

   char c = 'a';

2.2 2.变量的类型转换

(1)小类型的值会自动转换为大类型的值

   int a = 10;double b = 12.5;a + b-->double

(2)要把一个大类型的值存储到小类型中，需要进行强制类型转换，如果不转换，会编译出错

强制类型转换方式：(int)12.5-->把12.5这个double类型的转换为int型的

(3)当char,byte和short进行运算时，会自动转换为int型

   byte b1 = 12;byte b2 = 32;b1 + b2-->自动转为int型

(4)变量的作用域

   花括号内有效，在同一个作用域中不能重复定义变量。

       注释类型：

          //单行注释

          /*这是一个

          多行注释

          */

(5)定义变量的规定

①关键字用来定义变量

②对于类而言，定义名称的第一个字母用大写

对于函数或普通变量而言，第一个字母用小写，之后遵循驼峰标识

(6)目前来看String比以上所有类型都要大

   int a = 10;

   int b = 20;

   String str = "30";

   a + str + b;-->String类型-->103020

   a + b + str;-->3030

3.3.2 (2)选择结构

if(条件1) {

   满足条件1，执行这段代码;

} else if(条件2) {

   满足条件2，执行这段代码;

3.3.3 (3)分支结构

switch(整型或char型){

   case:xxx;//case后面直接是值，不能是逻辑表达式

   break;

   case:xxx;

   break;

   default:xxx;

   break;//每个case后需要加上break

}

3.3.4 (4)循环结构

①while()循环

while(条件) {

   条件满足一直执行这段代码，直到条件不满足;

}

②for循环

for(初始化;循环条件;单次循环完成后执行的代码) {

   循环体;

}

   continue;终止本次循环，跳回条件判断的地方

   break;结束整个循环

③do...while循环

do{

   初始化;

} while (条件) {

   循环体;

}
